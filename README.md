# Website Business Scooter Repo

This repository contains the current project workspace, including the scooter business landing page.

## Main website files

- `landing-page/index.html`
- `landing-page/styles.css`
- `landing-page/script.js`
- `landing-page/server.mjs`

## Local preview

Run:

```bash
npm run landing-page:start
```

Then open:

`http://127.0.0.1:4173`
